memcached-stat
--------------

Inspired by [Goでmemcachedのstatsを取得する](http://blog.restartr.com/2014/04/21/golang-memcache-stats-client/).

Install
-------

```
go get github.com/futoase/memcached-stat
```

Usage
-----

```
> memcached-stat
> memcached-stat -address="localhost:12345" 
```
